Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  1. Externe stylesheet
  2. Interne stylesheet
  3. Inline style

  1. Je kunt in een apart bestand de stijldefinities vermelden en deze opslaan met de extensie .css , deze zet je vervolgens in het desbetreffende HTML bestand in head zoals hier < link rel="stylesheet" href="naamvanhetdocument.css" type="text/css">
  2. Bij een interne stylesheet maak je de code in het HTML bestand tussen de head tags bijvoorbeeld zoals in het voorbeeld hieronder.

    3. < style type="text/css">
    < !--
    body {
    color: #3366FF;
    }

    h1{
    color: #CC0000;
    background-image: url(PATROON.GIF);
    padding: 20px;

    }
    pre {
    background-color: #ccffcc;
    margin: 5px;
    color: #000000;
    padding: 15px;
    margin-left: 15px;
    margin-right: 50px;
    border-top: 8px ridge #00CC33;
    border-right: 8px ridge #009966;
    border-bottom: 8px ridge #00cc33;
    border-left: 8px ridge #009933;
    }
    -->
    < /style >

    3. Je kunt zelfs in een enkele HTML-tag de stijl met CSS bepalen, bijvoorbeeld in de volgende P-tag: < P STYLE="color: red;">De tekstkleur in deze paragraaf is rood.

    Dit gebruik je als je incidenteel CSS wilt toepassen.
    (Merk op dat STYLE hier een attribuut is geworden -bij P- in tegenstelling het voorgaande voorbeeld waar STYLE een tag is.) Omdat je in de tag zelf de stijl definieert, zijn de vormgeving en de inhoud niet gescheiden. Wijzigingen van de vormgeving worden zo onoverzichtelijk en moeizaam. De kans op inconsequentie in de vormgeving neemt toe.
