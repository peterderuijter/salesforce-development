Er zijn verschillende soorten type Selectors, maak voorbeelden van type selectors, class selectors en descendant selectors

type selector.
Bijvoorbeeld: < style>
h1 {
color : red;
}

deze selector krijgen alle h1 tags automatisch een rode tekst.

Voor de class selector, zie hieronder:
< style >
.auto {
color : zwart;
}
Alle < p class="auto" > dan is deze tekst zwart< / p > dals je bv niet alle tags dezelfde styling wil hebben.

Deze selecteert alle elementen die afstammen van een bepaald element zoals je zit in dit voorbeeld worden alle < p> -elementen binnen < div> -elementen geselecteerd:
div p { background-color: yellow; }
je kunt dit aan meerdere elementen koppelen.
