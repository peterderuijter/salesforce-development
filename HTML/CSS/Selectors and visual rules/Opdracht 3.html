Met een type selector worden alle elementen van hetzelfde type, geselecteerd. Bijv: 

        /* Alle < P > elementen worden groen weergegeven. */
        p {
          color: green;
        }
Een descendant selector doelt op alle onderliggende elementen van een bepaald parent element. Bijv: 

        /* Alle elementen die binnen een < div> parent-element vallen worden groen weergegeven. */
        div span {
          color: green;
        }
Het attribuut class kan je gebruiken om verschillende elementen onder een zelfgekozen categorie te hangen, zoals bijvoorbeeld sociale media links, productnamen, de datum van een artikel, ... Het grote voordeel van classes is dat je ze kan hergebruiken op verschillende plaatsen en zelfs verschillende pagina's. Het wordt dan eenvoudig op CSS stijlregels toe te passen op deze klassen en ze een mooie opmaak te geven! 

            .social media  {
                background-color: green;
                color: white;
              }
